Typowe problemy pralek Candy w Maszewie

Najczęstsze awarie

  • Pralka Candy nie wiruje
  • Brak odpompowania wody
  • Nie grzeje wody
  • Hałas podczas wirowania
  • Pralka nie pobiera wody
  • Uszkodzona blokada drzwi
  • Pralka głośno odciska
  • Przecieki wody spod pralki
  • Pralka nie startuje

Kody błędów Candy

  • E01 / F1 – Problem z drzwiami / blokadą
  • E02 – Problem z dopływem wody
  • E03 – Problem ze zleceniem wody / odpływem
  • E05 / F16 – Błąd nagrzewania wody
  • E07 / E09 / F17 – Problem z silnikiem
  • E12 / E13 / F15 – Błąd modułu elektronicznego
