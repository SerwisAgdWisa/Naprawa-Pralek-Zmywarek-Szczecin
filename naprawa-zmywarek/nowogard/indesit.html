Typowe problemy zmywarek Indesit w Nowogardzie

Najczęstsze awarie

  • Zmywarka Indesit nie pobiera wody
  • Nie wypompowuje wody
  • Nie grzeje wody
  • Zmywanie zatrzymuje się w połowie cyklu
  • Przecieka woda spod zmywarki
  • Hałas lub wibracje podczas pracy
  • Problem z zamykaniem drzwi
  • Zmywarka nie uruchamia się

Kody błędów Indesit

  • F01 / E01 – Problem z grzałką / nagrzewaniem wody
  • F02 / E02 – Awaria elektroniki sterującej
  • F03 / E03 – Nieprawidłowy poziom wody / czujnik NTC
  • F04 / E04 – Problem ze spustem wody / pompa
  • F05 / E05 – Niedostateczny dopływ wody / blokada zaworu
  • F06 / E06 – Czujnik drzwi / drzwi nie zamknięte
  • F07 / E07 – Problem z silnikiem / pompą cyrkulacyjną
  • F08 / E08 – Awaria czujnika przepływu wody
  • F09 / E09 – Awaria czujnika temperatury
  • F10 / E10 – Wykryto przeciek wody
